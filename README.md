# RuuviTag experiments

This repository contains various RuuviTag experiments.

# Installation
```
sudo apt install bluez bluez-hcidump
mkvirtualenv -p /usr/bin/python3 ruuvi
pip install ruuvitag_sensor
```

# Testing
